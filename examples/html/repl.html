Connecting

To connect, make sure that both the XMPP server and BOSH server are running. Enter the full JID (e.g. name@example.net) and password you would normally use to connect to the XMPP server.

Queries

The query syntax is based on XPath 2.0. Lowercase words are interpreted as names (e.g. "/news" returns the item named "news" that is a child of the root). Uppercase words are interpreted as types (e.g. "//Folder" returns all Folders). By default, a non-absolute path is resolved relative to the root, so "/news" and "news" return the same thing.

Here are some example queries:

.The current item.
..The parent item.
*Children of the current item.
//.Everything.
newsThe items named "news".
news/PageChildren of "news" that are Pages.
news/descendant::*All descendants of "news".
//.[@description]Anything with a "description" property.
